Project Title
React Admin Dashboard

https://admin-dasvyh.netlify.app/

A modern, responsive admin dashboard built with React, Tailwind CSS, and Syncfusion components.
It includes interactive charts, tables, and a fully functional dark/light mode.

Features

📊 Dynamic Data Visualization – interactive charts, analytics, and tables

🌓 Dark & Light Mode toggle using Context API

🧩 Modular Components – reusable UI blocks (Navbar, Sidebar, Header, etc.)

🗂 Data Management – editable tables with filtering, sorting, and exporting

📱 Responsive Layout – works on desktop, tablet, and mobile

⚡ Syncfusion UI Library – powerful grid, chart, and calendar components


Built With

React

Tailwind CSS

Syncfusion React Components

React Context API

React Router
